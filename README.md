## アプリケーションの概要

簡易的な Todo リストです。  
React、テストコードを書く練習のために作成。

2022.05.12  
Express で API 開発して組み込み。  
バックエンド：https://github.com/shoco55/todo-api  
＊タスク追加や削除など、自由にお試しください。  
　データはデータベースに保存されますことをご理解いただいた上でご利用ください。
